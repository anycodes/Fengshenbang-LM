目前已经开源了杨戬系列模型和雷震子系列模型，后续还会陆续开源姜子牙、金吒、木吒、哪吒、申公豹、比干、增长天王（魔礼青）、持国天王（魔礼海）、多闻天王（魔礼红）、广目天王（魔礼寿）等系列。
